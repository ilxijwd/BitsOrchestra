# How to start back-end

`npm i -g serve`

`cd back-end/wwwroot`

`serve -s -p 8080`

`open http://localhost:8080/`

# How to start front-end

`cd front-end`

`pnpm install` (npm, yarn, etc.)

`cd fake-api`

`npx json-server fake-api/db.json`

`cd ..`

`pnpm run dev`
